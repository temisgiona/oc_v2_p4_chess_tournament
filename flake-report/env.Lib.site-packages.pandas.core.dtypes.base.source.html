39 See Also
40 --------
41 extensions.register_extension_dtype: Register an ExtensionType

45 Notes
46 -----
47 The interface includes the following abstract methods that must
48 be implemented by subclasses:
49  
50 * type
51 * name
52 * construct_array_type
53  
54 The following attributes and methods influence the behavior of the dtype in
55 pandas operations
56  
57 * _is_numeric
58 * _is_boolean
59 * _get_common_dtype
60  
61 The `na_value` class attribute can be used to set the default NA value
62 for this type. :attr:`numpy.nan` is used by default.
63

65 a default implementation, which relies on the ``_metadata`` class
66 attribute. ``_metadata`` should be a tuple containing the strings
67 that define your data type. For example, with ``PeriodDtype`` that's
68 the ``freq`` attribute.
69  
70 **If you have a parametrized dtype you should set the ``_metadata``
71 class property**.
72

74 parameters to your ``ExtensionDtype.__init__`` (if any). If any of
75 the attributes in ``_metadata`` don't implement the standard
76 ``__eq__`` or ``__hash__``, the default implementations here will not
77 work.
78

80 can be implemented: this method receives a pyarrow Array or ChunkedArray
81 as only argument and is expected to return the appropriate pandas
82 ExtensionArray for this dtype and the passed values::
83

85

87 self, array: Union[pyarrow.Array, pyarrow.ChunkedArray]
