84 """
85 A monkeypatch to tells pandas to let us in.
86  
87 By default, passing a PandasArray to an index / series / frame
88 constructor will unbox that PandasArray to an ndarray, and treat
89 it as a non-EA column. We don't want people using EAs without
90 reason.
91  
92 The mechanism for this is a check against ABCPandasArray
93 in each constructor.
94  
95 But, for testing, we need to allow them in pandas. So we patch
96 the _typ of PandasArray, so that we evade the ABCPandasArray
97 check.
98 """

100 m.setattr(PandasArray, "_typ", "extension")
