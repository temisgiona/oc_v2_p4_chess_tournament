50 Set the labels at the center of the window.

64

67 rolling : Provides rolling window calculations.

69  
70 Notes
71 -----
72 By default, the result is set to the right edge of the window. This can be
73 changed to the center of the window by setting ``center=True``.
74  
75 Examples

77 >>> df = pd.DataFrame({"B": [0, 1, 2, np.nan, 4]})

84 4 4.0
