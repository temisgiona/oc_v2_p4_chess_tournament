38 class TCPServer(object):
39 r"""A non-blocking, single-threaded TCP server.
40  
41 To use `TCPServer`, define a subclass which overrides the `handle_stream`
42 method. For example, a simple echo server could be defined like this::
43  
44 from tornado.tcpserver import TCPServer
45 from tornado.iostream import StreamClosedError
46 from tornado import gen
47  
48 class EchoServer(TCPServer):
49 async def handle_stream(self, stream, address):
50 while True:
51 try:
52 data = await stream.read_until(b"\n")
53 await stream.write(data)
54 except StreamClosedError:
55 break
56  
57 To make this server serve SSL traffic, send the ``ssl_options`` keyword
58 argument with an `ssl.SSLContext` object. For compatibility with older
59 versions of Python ``ssl_options`` may also be a dictionary of keyword
60 arguments for the `ssl.wrap_socket` method.::
61  
62 ssl_ctx = ssl.create_default_context(ssl.Purpose.CLIENT_AUTH)
63 ssl_ctx.load_cert_chain(os.path.join(data_dir, "mydomain.crt"),
64 os.path.join(data_dir, "mydomain.key"))
65 TCPServer(ssl_options=ssl_ctx)
66  
67 `TCPServer` initialization follows one of three patterns:
68  
69 1. `listen`: simple single-process::
70  
71 server = TCPServer()
72 server.listen(8888)
73 IOLoop.current().start()
74  
75 2. `bind`/`start`: simple multi-process::
76  
77 server = TCPServer()
78 server.bind(8888)
79 server.start(0) # Forks multiple sub-processes
80 IOLoop.current().start()
81  
82 When using this interface, an `.IOLoop` must *not* be passed
83 to the `TCPServer` constructor. `start` will always start
84 the server on the default singleton `.IOLoop`.
85  
86 3. `add_sockets`: advanced multi-process::
87  
88 sockets = bind_sockets(8888)
89 tornado.process.fork_processes(0)
90 server = TCPServer()
91 server.add_sockets(sockets)
92 IOLoop.current().start()
93  
94 The `add_sockets` interface is more complicated, but it can be
95 used with `tornado.process.fork_processes` to give you more
96 flexibility in when the fork happens. `add_sockets` can
97 also be used in single-process servers if you want to create
98 your listening sockets in some way other than
99 `~tornado.netutil.bind_sockets`.
100  
101 .. versionadded:: 3.1
102 The ``max_buffer_size`` argument.
103  
104 .. versionchanged:: 5.0
105 The ``io_loop`` argument has been removed.
106 """

219  
220 If num_processes is ``None`` or <= 0, we detect the number of cores
