15 class ConEmuOutput:
16 """
17 ConEmu (Windows) output abstraction.
18  
19 ConEmu is a Windows console application, but it also supports ANSI escape
20 sequences. This output class is actually a proxy to both `Win32Output` and
21 `Vt100_Output`. It uses `Win32Output` for console sizing and scrolling, but
22 all cursor movements and scrolling happens through the `Vt100_Output`.
23  
24 This way, we can have 256 colors in ConEmu and Cmder. Rendering will be
25 even a little faster as well.
26  
27 http://conemu.github.io/
28 http://gooseberrycreative.com/cmder/
29 """
