171 class LocalStack:
172 """This class works similar to a :class:`Local` but keeps a stack
173 of objects instead. This is best explained with an example::
174  
175 >>> ls = LocalStack()
176 >>> ls.push(42)
177 >>> ls.top
178 42
179 >>> ls.push(23)
180 >>> ls.top
181 23
182 >>> ls.pop()

184 >>> ls.top
185 42

188 the :func:`release_local` function but the correct way is to pop the
189 item from the stack after using. When the stack is empty it will

193 the topmost item on the stack.
