103 """
104 Concatenate pandas objects along a particular axis with optional set logic
105 along the other axes.
106  
107 Can also add a layer of hierarchical indexing on the concatenation axis,
108 which may be useful if the labels are the same (or overlapping) on
109 the passed axis number.
110  
111 Parameters
112 ----------
113 objs : a sequence or mapping of Series or DataFrame objects
114 If a mapping is passed, the sorted keys will be used as the `keys`
115 argument, unless it is passed, in which case the values will be
116 selected (see below). Any None objects will be dropped silently unless
117 they are all None in which case a ValueError will be raised.
118 axis : {0/'index', 1/'columns'}, default 0
119 The axis to concatenate along.
120 join : {'inner', 'outer'}, default 'outer'
121 How to handle indexes on other axis (or axes).
122 ignore_index : bool, default False
123 If True, do not use the index values along the concatenation axis. The
124 resulting axis will be labeled 0, ..., n - 1. This is useful if you are
125 concatenating objects where the concatenation axis does not have
126 meaningful indexing information. Note the index values on the other
127 axes are still respected in the join.
